REQUISITOS PREVIOS:
  • Python 3.8+
  • pip (gestor de paquetes)
  • Soberanía operativa local (sin dependencias externas)
  • Linux/Unix (recomendado para máxima compatibilidad)

PASOS DE INSTALACIÓN:

1. PREPARACIÓN DEL ENTORNO:
   $ cd /home/gus/ECN
   $ python3 -m venv .venv
   $ source .venv/bin/activate

2. INSTALACIÓN DE DEPENDENCIAS:
   $ pip install --upgrade pip
   $ pip install flask
   $ pip install python-socketio
   $ pip install python-engineio
   $ pip install pynput (para HID)
   $ pip install pyyaml
   $ [otras dependencias según necesidad]

3. VALIDACIÓN DEL MÓDULO_INTEGRADOR:
   $ python3 NEUROBIT_SOS_FOLDER_MODULES/adapters.py --test

4. INICIALIZACIÓN DE ESTRUCTURA:
   $ mkdir -p data/fragments
   $ mkdir -p data/bitacora
   $ mkdir -p storage/backups

5. INICIAR AWAKE_CEREMONY:
   $ python3 awake/awake_ceremony.py

6. INICIAN SERVICIOS BASE:
   • Servidor Flask en puerto 5001
   • PROCESS_BASELINE comienza registro
   • NEUROBIT_API disponible

7. INICIAR GUI PRINCIPAL:
   $ python3 interface/neurobit_central_station_gui.py

8. MONITOR EN TIEMPO REAL:
   Acceder a: http://localhost:5001/salon_debug.html

  $ python3 NEUROBIT_SOS_FOLDER_MODULES/adapters.py --validate

  $ python3 core/sos_file_manager.py --backup

  $ python3 core/sos_file_manager.py --restore <backup_id>
